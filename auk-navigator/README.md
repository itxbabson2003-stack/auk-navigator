# AUK Navigator

## Overview
AUK Navigator is a web-based application designed to help users navigate the Al-Qalam University campus using an interactive satellite map. The application allows users to set their current location, select destinations, and view routes on the map.

## Features
- Interactive satellite map for enhanced visual experience.
- User-friendly interface with improved design and layout.
- Ability to set "You are here" location by tapping on the map.
- Route planning functionality to get directions between locations on campus.
- Search functionality for quick access to campus locations.
- Responsive design for optimal viewing on various devices.

## Setup Instructions
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd auk-navigator
   ```
3. Open `index.html` in your web browser to view the application.

## Usage
- Tap anywhere on the map to set your current location.
- Select a destination from the dropdown or search bar.
- Click on "Get Directions" to view the route on the map.
- Use the "Clear Route" button to reset your selections.
